YOUR INFRASTRUCTURE
Sentence segments + hashes
a3f2 Hierbij verwijs ik Jan de Vries ([ID_1]) naar uw afdeling cardiologie.
b7c1 De patiënt is bereikbaar via [PHONE_1].
d9e4 Jan heeft ook een wijziging van begunstigde gevraagd.
WHAT HAPPENED
Text is split at sentence boundaries. Each segment gets a SHA-256 hash (shown as first 4 chars).
The SDK keeps a local hash → position mapping so it can reassemble later. The model only sees the hash, never the position or document structure.

WHAT HAPPENED
The AI returns JSON annotations only — the matched text, entity type, and confidence level. It never reprints or echoes the input text.
